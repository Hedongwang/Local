首先是安装git软件，基本上是默认安装。
安装成功后，使用cd ,mkdir命令（git命令中，有好多是跟Linux很像的，只是加入了git的前缀而已）进入你想设置的本地仓库中。
随后，使用
	git config --global user.name "你的github用户名"
	git config --global user.email "你的GitHub邮箱（2358661873@qq.com）"
如果想要上传某个代码文件，则将上传的文件复制到当前的本地仓库中，使用git add .代码将文件暂存到工作区中，如果上传则是git commit -m"对于本次上传的说明" 来将文件上传到本地仓库中。
随后开始与远程仓库的连接：输入cd ~/.ssh/（Windows中的~下的文件就是C:\Users\Administrator\.ssh），如果显示文件夹无效，则输入mkdir ~/.ssh 回车，再输入ssh-keygen -t rsa -C "邮箱（2358661873@qq.com）"后不断回车，直到出现一个新的输入符号.
此时就打开C:\Users\Administrator\.ssh文件，找到一个pub结尾的文件，打开复制，再打开GitHub中设置中的ssh，添加一个ssh秘钥，就完成了远程的连接。
打开一个GitHub上的远程仓库，复制此时的网址，输入git remote add origin 网址复制上去，最后用一条代码：git push -u origin master就将本地仓库中的文件上传到远程仓库中去了。
